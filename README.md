## :cd: Installation

```sh
git clone git@github.com:filipe1309/rs-kafka-node-certification.git
```

```sh
cd rs-kafka-node-certification
```

## :runner: Running

**Up Kafka container**

```sh
docker-compose up
```

**Up API**

```sh
cd api
yarn dev
```

**Up Node.js Certification Microservice**

```sh
cd certification
yarn dev
```

**Call the API**

```sh
POST http://localhost:3333/certifications
```
